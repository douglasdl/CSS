# CSS

## Normal Flow
- **block:** uses the whole line. The next element will go to the next line.
- **inline:** uses only the required width for the the component in the line. The next element will keep in the same line.


## Table
```html
<table>
  <tr>
    <td>A</td>
    <td>B</td>
  </tr>
  <tr>
    <td>1</td>
    <td>2</td>
  </tr>
</table>  
```

## Tableless
- **float:** left, right.
- **clear:** both.
